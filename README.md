# 多功能智能助手平台

> 一个网页入口，集成 6 个 AI 小工具的对话式助手平台。
> 零基础项目实训作品（6/1 – 6/5）· 框架 **Streamlit** · 模型 **智谱 GLM**

一个把"输入→输出"的工具，做成**对话式 + 结果可操作**体验的 AI 工具集：
每个工具都能连续追问、一键调整语气、重新生成、下载结果；PDF 工具支持
**带页码引用的问答**与**结构化抽取**。无 API key 也能完整演示（双模式兜底）。

---

## 亮点

- **对话式交互**：每个工具都是多轮对话，可基于上一轮继续改，不是一次性表单
- **结果可操作**：重新生成 / 下载 / 复制 / 快捷追问（更正式、更简短、换语气…）
- **双模式兜底**：未配 key 或 API 不通时自动回退「模拟返回」，**演示永不崩**
- **演示安全模式**：一键强制走模拟，瞬间出结果、零网络风险，适合彩排与验收
- **PDF 做深**：按页标记 → 回答带「（见第 3 页）」页码引用 + 一键抽取成表格
- **纯本地工具**：CSV 预览完全离线运行，断网也能演示
- **工程素养**：模块解耦、`pytest` 单元测试、历史截断控制成本、密钥不入库

## 快速开始

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2.（可选）配置 API key —— 不配也能跑，自动走模拟返回
cp .env.example .env        # 然后编辑 .env 填入你的智谱 key

# 3. 启动
streamlit run app.py
```

浏览器自动打开 `http://localhost:8501`，左侧边栏切换工具。

`.env` 内容：

```
ANTHROPIC_BASE_URL=https://open.bigmodel.cn/api/anthropic
ANTHROPIC_AUTH_TOKEN=你的智谱key
LLM_MODEL=glm-4.6
```

> 使用智谱 GLM 的 Anthropic 兼容端点。`.env` 已被 `.gitignore` 忽略，不会进仓库。

## 架构

```
                         ┌─────────────┐
       浏览器  ────────▶ │   app.py    │  st.navigation 导航 + 线性图标
                         └──────┬──────┘
                                │
               ┌────────────────┼────────────────┐
               ▼                ▼                 ▼
        views/*.py         chat_tool.py        ui.py
        各工具页面     对话组件(气泡/追问/      安全模式开关
        (调用组件)      重生成/下载/截断)        提示词查看
               │                │
               ▼                ▼
          prompts.py       llm_client.py ──────▶ 智谱 GLM API
          提示词集合     双模式(真实/模拟)+流式      (Anthropic 兼容)
                                │
                          pdf_utils.py
                       PDF 分页抽取+页码标记
```

设计要点：
- **关注点分离**：界面（views）、交互组件（chat_tool）、模型客户端（llm_client）、
  纯逻辑工具（pdf_utils）各司其职，便于测试与维护。
- **失败优雅降级**：所有模型调用经 `llm_client`，异常统一回退模拟返回。
- **成本可控**：对话历史截断（`MAX_TURNS`），PDF 正文按字符预算截断。

## 常见问题

- **没装 Python？** 需要 Python 3.10+，`python --version` 检查。
- **AI 工具显示"模拟返回"？** 没配 key、开了安全模式、或 API 偶发不可用时的正常兜底。
- **智谱端点偶发 503？** 共享端点拥挤所致，多点几次「重新生成」即可，或开「演示安全模式」。
- **CSV 中文乱码？** 已自动尝试 utf-8 / gbk 两种编码。
- **扫描版 PDF 没文本？** 图片型 PDF 需 OCR，本工具暂不支持。
