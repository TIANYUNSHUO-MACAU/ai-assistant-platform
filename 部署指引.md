# 部署上线指引（Streamlit Community Cloud）

把这个项目变成一个**别人能点开的公网链接**。Streamlit Community Cloud 免费，最适合本项目。

## 前提

- 一个 GitHub 账号
- 代码已推到 GitHub 仓库（**注意：`.env` 不会被推上去，这是对的**）

## 步骤

### 1. 把代码推到 GitHub

```bash
# 在 GitHub 上新建一个空仓库，拿到地址后：
git remote add origin https://github.com/你的用户名/你的仓库名.git
git branch -M main
git push -u origin main
```

> 已有 `.gitignore` 保证 `.env`、`secrets.toml` 不会上传，密钥安全。

### 2. 在 Streamlit Cloud 部署

1. 打开 https://share.streamlit.io ，用 GitHub 账号登录
2. 点 **New app** → 选择你的仓库、分支 `main`、主文件 `app.py`
3. 点 **Deploy**

### 3. 配置密钥（关键，否则只会走模拟返回）

部署后进入 App 的 **Settings → Secrets**，粘贴（参考 `.streamlit/secrets.toml.example`）：

```toml
ANTHROPIC_BASE_URL = "https://open.bigmodel.cn/api/anthropic"
ANTHROPIC_AUTH_TOKEN = "你的智谱key"
LLM_MODEL = "glm-4.6"
```

保存后 App 会自动重启，几十秒后真实模型即可生效。

> 代码已做兼容：本地读 `.env`，云端读 Secrets，同一份代码两处都能跑
> （见 `llm_client.py` 的 `_cfg` 函数）。

## 上线后注意

- **偶发"模拟返回"**：智谱共享端点拥挤会 503，属正常。可在工具页打开
  「演示安全模式」稳定演示，或多点几次「重新生成」。
- **隐私提示**：PDF / 文本内容会发送到智谱 API。演示私密文档时请注意。
  CSV 预览是纯本地，不外发。
- **休眠**：免费版 App 长时间无访问会休眠，下次访问自动唤醒（稍慢几秒）。

## 验收用

部署成功后，把公网链接写进 README 顶部和验收材料，直接满足
PPT 验收门槛里的「仓库链接可访问 / Demo 可访问」。
