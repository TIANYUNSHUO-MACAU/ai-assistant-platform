# 测试清单（验收逐项打勾）

> 对应 PPT 第 12 页验收门槛。演示前按此清单逐项确认。

## 基础线（必须完成）

- [ ] 仓库链接可访问（已 push 到 GitHub / 云效）
- [ ] `pip install -r requirements.txt` 能成功安装
- [ ] `streamlit run app.py` 能启动，浏览器打开首页
- [ ] 左侧边栏能看到 4 个工具页面
- [ ] 至少 1 个功能完整闭环（输入 → 有可解释输出）
- [ ] 每人能说明自己负责的部分

## 目标线（尽量争取）

- [ ] 能讲清楚为什么选 Streamlit（见 `框架选型说明.md`）
- [ ] 2-3 个以上工具可用并集成在同一入口
- [ ] 有本测试清单 + 已知问题记录
- [ ] 演示时能说明取舍（双模式、放弃前后端分离等）
- [ ] 提示词集合可复用（见 `prompts.py`）

## 逐功能测试

### 文案生成
- [ ] 输入场景，点击生成，返回多条文案（真实或模拟）
- [ ] 空输入时有提示，不报错

### 翻译助手
- [ ] 输入中文 → 返回英文
- [ ] 输入英文 → 返回中文
- [ ] 空输入时有提示

### 简历优化
- [ ] 输入简历片段，返回问题诊断 + 优化版本
- [ ] 空输入时有提示

### CSV 预览（断网也要能过）
- [ ] 上传一个 CSV，显示数据预览
- [ ] 显示行数/列数/缺失值
- [ ] 显示各列类型说明
- [ ] 含中文的 CSV（gbk 编码）也能正确读取

- 模拟模式下 AI 工具返回的是占位文本，仅用于演示流程，非真实结果。
- 智谱模型名以实测可用为准（当前配置：见 .env 的 LLM_MODEL）。
